# Features List

- **Detects wasteful spending patterns** using deep AI textual classification of raw transactions.
- **Calculates long-term financial loss** by simulating compound interest opportunity costs (Money Mirror).
- **Generates behavioral insights** via aggressive, no-nonsense financial feedback.
- **Simulates financial commitment system** using an Ethereum escrow layer.
- **Creates social accountability** through a live, highly competitive community leaderboard and activity feed.
