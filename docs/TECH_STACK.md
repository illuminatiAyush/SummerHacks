# Tech Stack Reference

## Frontend
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS + Framer Motion
- **State Management:** Zustand, LocalStorage

## Backend
- **Framework:** FastAPI
- **Language:** Python
- **Functionality:** Proxies requests, sanitizes strings, drives logic generators.

## AI Engine
- **Orchestration:** LangGraph
- **LLM Provider:** Groq
- **Design Pattern:** Prompt Chaining with output enforcement.

## Data Integration
- **Framework:** Setu AA Sandbox (Account Aggregator framework)

## Web3 Protocol
- **Language:** Solidity
- **Network:** Ethereum (Sepolia Testnet)
- **Tooling:** Ethers.js, MetaMask Plugin (Optional)
